# 🎬 CineScope – Movie Discovery App

CineScope is a modern **React + TypeScript** web application that allows users to discover movies using the **TMDB API**.  
It features a clean UI, responsive design, and interactive movie cards inspired by Netflix.

## ✨ Features

- 🔍 Real-time movie search with debounced input  
- 🎬 Browse trending and popular movies (TMDB API)  
- 🃏 Interactive movie cards with dynamic data  
- 📄 Expandable movie details (overview view)  
- 📊 Trending search analytics powered by Appwrite  
- 📱 Fully responsive UI (mobile, tablet, desktop)  
- ⚡ Optimized API fetching with caching logic  
- 🎨 Netflix-inspired modern UI design  
- 🚀 Fast and smooth user experience  

---

## 🛠️ Tech Stack

- ⚛️ React + TypeScript  
- ⚡ Vite (Build Tool)  
- 🎨 Tailwind CSS  
- 🎬 TMDB API  
- 🧩 Appwrite (Backend Analytics)  
- 🧱 Material UI (Loaders)
