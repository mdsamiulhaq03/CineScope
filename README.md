# 🎬 CineScope – Movie Discovery App

CineScope is a modern React + TypeScript web application that allows users to discover movies using The Movie Database (TMDB) API. It features a clean UI, responsive design, and interactive movie cards similar to Netflix.

## ✨ Features

- 🔍 Real-time movie search
- 🎬 Browse popular movies (TMDB API)
- 🃏 Interactive movie cards
- 📄 Toggle movie details (overview)
- 📱 Fully responsive design
- ⚡ Fast API-based data fetching
- 🎨 Netflix-style UI layout

---

## 🛠️ Tech Stack

- React + TypeScript
- Tailwind CSS
- TMDB API
- Material UI (Loader)
- Vite
